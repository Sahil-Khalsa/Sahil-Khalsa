## About

I am a Software Development Engineer who builds scalable backend services, distributed workflows, and cloud native systems, with deep hands on experience shipping production grade AI for real users.

My work spans four areas:

- **Software Engineering.** I design REST APIs, microservices, schema backed data models, and reliable production services with monitoring, debugging, and CI/CD built in from day one.
- **AI and ML Engineering.** I build LangGraph agents, multimodal RAG pipelines, ColPali style visual retrieval, deterministic validators, and LLM evaluation suites that hold up under adversarial testing.
- **Full Stack Development.** I ship end to end with React, Next.js, FastAPI, Django REST, and Node.js across public and protected workflows.
- **Product Engineering.** I translate ambiguous requirements into shipped features, own reliability end to end, and treat latency, cost, and correctness as first class metrics.

Across recent roles I have scaled a multi service platform, secured over a million generated assets, cut P95 latency by more than half, and delivered AI systems that reach 92 percent grounded accuracy in evaluation.

> **Open to:** Software Engineer, AI Engineer, Forward Deployed Engineer, and Full Stack roles.
